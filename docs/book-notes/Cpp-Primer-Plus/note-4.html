处理数据

# 简单变量

以两个下划线或下划线和大写字母打头的名称被保留给编译器实现；以一个下划线开头的名称保留给全局变量。

climits 头文件保留了各种变量的范围，也可以通过 sizeof 来查看各种变量的字节位数。

#include <iostream>
#include <climits>

int main()
{
    using namespace std;
    int n_int = INT_MAX;
    short n_short = SHRT_MAX;
    long n_long = LONG_MAX;
    
    cout << "int is" << sizeof(int) << " bytes." <<endl;
    cout << "max int is " << n_int;
}

通过 hex, oct, dec 来控制输出格式。如果要控制常数格式，需要加后缀。比如 1000ul, 232ull 等。

# 宽字符

定义了 wchar_t 类型，用于处理宽字符，注意这还是 8 比特的变量，如果要定义双字节或者四字节的变量，需要使用 char16_t 或者 char32_t 。

wchar_t name = L'c';
wcout << L'chen' << endl;

char16_t name = u'\uchen';
char32_t name = U'\U0000abcd';

#include <iostream>

int main()
{
    using namespace std;
    cout.setf(ios_base::fixed, ios_base::floatfield);
    float tub = 10.0 / 3.0;
    double mint = 10.0 / 30.0;
    const float million = 1.0e6;

    cout << tub << endl;

# 隐式类型转换

C++ 中的基本类型提升有以下两种

  • 整形提升或类型提升 ：对于 bool,char,unsigned char,short,unsigned short 若上述类型的所有可能的值都能包含在 int 内（即能表示数的范围包含了前面类型能表示的范围），那么上述类型在进行算术运算是就会转换为 int 类型，否则，它们会转换为 unsigned int 类型。
  • 标准转换或类型转换： long double,double,float,unsigned long long,long long,unsigned long,long,unsigned int,int 之间的转换。
  • 在任何涉及两种数据类型的操作中，它们之间等级较低的类型会被转换成等级较高的类型。

# 强制类型转换

两种格式，尽量使用第二种，同时 C++ 还引入了 4 个强制类型转换符号。

(typename) value    // C
typename (value)    // C++

static_cast<long> (value)

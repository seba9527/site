# 题目:bof - 5 pt

Nana told me that buffer overflow is one of the most common software vulnerability. Is that true? 娜娜告诉我缓冲区溢出是最常见的软件漏洞之一，这是真的么？

Download : http://pwnable.kr/bin/bof

Download : http://pwnable.kr/bin/bof.c

Running at : nc pwnable.kr 9000

# 4月1日

# 上午

早上晨跑了。调试了一下那个shellcode的过程，没问题，估计是offset算错了，还是什么其他问题。然后开了组会，师姐认为我双机调试目标不明确，首先应该分析ring3的进程入口，其次再调试kernel，很有道理。同时黄老师说经费充足，电缆可以购买。

# 下午

关注了一波GitHub上的996，害怕。学习期间996没什么，以后工作谁想996啊，看了知乎很多大佬在分析，不知道说什么，围观吃瓜吧。然后看了师姐说的那个问题，查了一波Microsoft的资料，好像srv.sys不直接和ring3的程序进行通信，应该直接和os，感觉还是不行的，GG。

# 晚上

因为系统重装过，SSH和GPG都没了，花了点时间重新配置了Git，现在每次都能verify了。然后试了试Git新推出的LFS功能，很牛逼，但是带宽有限，而且push不到gitbook上面，遂放弃。之后随便看看发现HelloGitHub这个项目挺有意思的，还发现了一款英文报纸，南华早报SouthChinaMorningPost，看了感觉比BBC写的有意思，哈哈以后能多看看英文的新闻。

# 4月2日

# 上午

早上晨跑，哦耶。看论文，同时继续翻译毕设相关论文，然后查看了系统里的通知，时间不多了，抓紧搞毕设，我开始慌了。

# 下午

中午回去洗了衣服。然后继续翻译论文， 总算把固定要翻译的部分解决了。整理了一下来所里几个月的收获，开始写毕业实习的报告，烦，写了一些，不知道具体要求，我还是之后再写吧。之后开始准备把SCA的环境搭起来，一堆问题。吃晚饭前通知毕业实习报销不了，因为我们是分散实习的。我日。

# 晚上

晚上群里说还要填什么实习申请表，咨询了一波老张，老张表示不需要（稳）。然后各种问实习的问题，没通知到位真是烦。之后开始写实习报告了，没要求，随便写，省的之后再催我，烦，搞完拉倒。后来继续看毕设的论文和搭建环境，搞了一会又通知准备实习答辩，所以继续写实习报告。

# 4月3日

# 上午

早上睡过了，(lll￢ω￢)没起来跑步。然后继续写实习报告，准备今天把Golang逆向和MS17-010的部分写完，看看够不够。

# 下午

一直在写MS17-010的报告，4点半的时候终于搞完了，写得手快断了。

# 晚上

继续写实习报告，把第三部分准备好了，争取明天写完。8点就回去了，补早上的跑步。

# 4月4日

# 上午

早上跑步了，上午继续完成毕业实习的第三部分。烦

# 下午

下午继续写实习报告，手要断了。3点半的时候终于写完了。准备去签字和盖章。

# 晚上

晚上没做什么，摸鱼了，回去洗衣服。准备明天的比赛。

# 4月5日

# 上午

打比赛，肝一道base逆向。

# 下午

继续打比赛，肝一道SEH逆向和LCTF2018的逆向题。最后ak了逆向。

# 晚上

打不动pwn了，做了一天，累死了，准备回去了。最后拿了第一，听说有2000？不知道几个人分。

# 4月6日

# 上午

回顾了昨天打的比赛，然后又分析了其他的问题。

# 下午

打了其他的比赛，感觉很吃力，分析了几个re。感觉自己太菜了。

# 晚上

晚上很早就回去了，休息一个晚上。

# 4月7日

# 上午

上午在装虚拟机，虚拟机坏了，我真是日了VMware。然后和同学去吃了午饭，花了100一个人。

# 下午

中午2点才吃完，下午随便逛了逛，然后回宿舍继续看movfuscator，搭好了环境，最后调试也没出结果，gdb没法下硬件断点，然后用了ida，气死了，没做出来，自己还是太菜了。

# 晚上

晚上去跑了步，打了篮球。还是很舒服的。

# 4月8日

# 上午

上午开了组会，自己说了上周的进展，好像比较少，确实上周就调研了下ring3的问题，没实际干什么项目的事情，环境没搭起来我也很绝望啊。老师好像有点不满意。

# 下午

下午纠结了实习报告的事情，黄老师很忙，没来得及给他签字，(lll￢ω￢)，明天再找机会吧。然后月赛发了1000的奖金，舒服哈哈，后来通知我周末讲题，好吧，还得整理整理题目。难受。

# 晚上

一直在整理题目，没干其他的，把实习报告最后也弄完了。

# 4月9日

# 上午

上午继续整理wp，然后实习报告给老师签字了，之后收到通知说老师要来实习单位建查，我也是服了。和导师沟通了一下，事情真多。烦。

# 下午

下午在服务器上装Ubuntu18的os，然后发现显卡出了问题。很难受，不知道怎么办，然后把第一个题目的wp写完了。晚上开始整理第二个题目的wp。

# 晚上

在写wp，然后查了下系统为什么花屏的原因，可能是驱动的问题，但是重装没用，什么鬼。后来在出题捞金，大致想好了几个出题的思路，然后在纠结misc该出什么，想到了福尔摩斯里面跳舞的小人。

# 4月10日

# 上午

上午把wp都弄好了。

# 下午

屁事真多，不是写实习报告就是弄实习协议，盖章也盖不了。然后又通知说什么保研审核，我真是服了，屁事真多。

# 晚上

7点多回去的，测甲醛，期间出去跑步了。最后结果是甲醛超标2倍，难受啊。

# 4月11日

# 上午

上午在弄pintools环境。

# 下午

搞了一上午，pintools死活都不行，疯了。之后又换了n个不同的环境，一直在测试。

# 晚上

pintools实在不行，最后看了下code的需求。想到idc可能可以，然后根据doc开始搞idapython，觉得有戏。

# 4月12日

# 上午

上午老师来检查，接待了老师。然后继续看pintools，同时上传了毕设的翻译原文，译文也搞完了。今天终于把昨天的问题解决了，idapython大法好。

# 下午

下午的时候，今年DDctf开始了，花时间做了2个题，不得不说题目难度远不及去年啊，暗示滴滴不行了。不过后面几个题有点难度。

# 晚上

感觉后面那个mac的逆向不难，然后仔细一看是一个虚拟机，哇，真的简单，醉了。然后第四个逆向，全是宏指令，但是根据程序流程，自己硬是判断出来按位检测的方式，于是想到侧信道SCA攻击，祭出pin搞了一波，不出所料，就很舒服，还差一个就ak了。

# 4月13日

# 上午

打比赛，搞了那个pwn，额，整数溢出的问题，由编译器是gcc5以上的，所以还有一个检测的地方，通过打印出的ebp的值来泄露栈地址，从而泄露libc地址，然后常规做法。

# 下午

下午开始出题，哈哈。

# 晚上

在出easy level的逆向，预期解法是侧信道、angr或者idapython，出题真是痛苦，难受。

# 下午

# 上午

上午继续在出题，基本出好了，解题脚本也弄完了，测试了下angr的脚本，不知道为什么老是会卡住，release版本没问题，但是debug就一直会卡住。看了下是编译的指令不一样，emm，之后再看。

# 下午

# 晚上

# 下午

出题啊，一直在构思题目的做法，想到了虚拟机，然后开始设计虚拟机的指令集。

# 晚上

突然想到矩阵，然后联想到数独求解，准备写一个数独的校验函数。

# 4月17日

# 上午

数独部分写完了，本来想加一个离散对数的，但是要实现一个大数类，感觉太烦了。后来想想换成一次取余式用中国剩余定理搞好了。

# 下午

修理整改代码，全部换成了线程来处理，期间看了矿大的比赛，哎。然后又用STL模板库来替换低级的写法，最后用C++11又重写了部分代码。

# 晚上

晚上继续调了调代码，差不多ok了。最后又加上了反调试的部分，以及CRC校验的逻辑，基本好了。有空再优化下吧。
